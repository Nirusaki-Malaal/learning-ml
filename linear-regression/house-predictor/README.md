# Boston AI House Predictor 🏠

A predictive machine learning interface for estimating Boston housing market valuations using advanced Neural and Linear Regression models. Featuring a responsive, ultra-premium UI powered by Google Stitch's "Liquid Glass AI" and high-fidelity FastAPI backend.

## Project Structure 📁

```text
.
├── main.py                # Main FastAPI application entry point
├── requirements.txt       # Dependencies
├── data.csv               # Boston Housing raw dataset
├── rules.txt              # Guidelines for the modern UI/UX
├── plugins/               # Core Python modules
│   ├── house_predictor.py # Machine Learning regression model script
│   └── __init__.py
├── templates/             # HTML Templates
│   └── index.html         # Frontend interface, generated by Google Stitch
└── statics/               # Static Assets
    ├── style.css          # CSS styles with Glassmorphism 2.0 aesthetics
    └── app.js             # Client-side JavaScript for making API calls and interactivity
```

## Features ✨

- **Actual Boston Dataset**: Trained directly on the well-known Boston housing criteria (`CRIM`, `ZN`, `RM`, `LSTAT`, etc.).
- **Interactive UI**:
  - **Explainable AI Text**: Hovering over inputs provides real-time model interpretation.
  - **Random Load**: Test the application quickly by fetching a random subset of training data.
- **Modern "Liquid Glass" Visuals**: A frosted-glass motif created strictly according to 2026 UI trends.

## Setup & Execution 🚀

1. **Install Requirements**
   Make sure you have Python 3 installed, then install the required pip modules:
   ```bash
   pip install -r requirements.txt
   ```

2. **Start the Predictor Backend**
   Execute the FastAPI environment locally.
   ```bash
   uvicorn main:app --reload
   ```

3. **Access the Application**
   Open your browser and visit:  
   [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## Tech Stack 🛠️

- **Backend:** FastAPI, Pydantic, Uvicorn
- **Machine Learning:** Scikit-Learn (LinearRegression), Pandas
- **Frontend:** Vanilla JS, CSS (Glassmorphism aesthetics via generated Tailwind setup), Jinja2Templates
